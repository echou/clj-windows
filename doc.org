* Tasks

**   Prepare powershell for executing the scripts
 
to allow the execution of local scripts  for the current user the execution-policy has to be adjusted [fn:2]

#+BEGIN_SRC 
set-executionpolicy -Scope CurrentUser remotesigned
#+END_SRC
